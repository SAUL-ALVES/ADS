# Breve descrição do projeto

 Projeto introdutório com o objetivo de aplicar os conceitos fundamentais do React na construção de interfaces dinâmicas. A aplicação consiste em uma listagem de anúncios de carros exibidos em cards, utilizando componentização, props e hooks (useState). Também foi implementada uma funcionalidade de busca por modelo de carro, permitindo a filtragem dos dados em tempo real, simulando o comportamento de aplicações modernas baseadas em estado e interação do usuário.
