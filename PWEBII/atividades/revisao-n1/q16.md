Sobre métodos de iteração em arrays: Explique como funcionam os métodos de iteração em arrays em JavaScript: forEach(), map(), filter() e reduce().

R:

 O foreach funciona como uma estrutura de repetição que percorre todos os elementos de uma coleção (como uma lista, array ou objeto) uma vez por cada elemento, executando um bloco de código específico para cada um.
 
 O map retorna um novo array, do mesmo tamanho, onde cada elemento corresponde ao resultado da função aplicada.

 O filter retorna um novo array contendo apenas os elementos que passaram no teste (retornam true).

 O reduce reduz um array a um único valor, que pode ser número, string, array, objeto, etc..
